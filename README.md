# 3FabRec
PyTorch implementation of '3FabRec: Fast Few-shot Face alignment by Reconstruction' (CVPR2020)

![Teaser](images/teaser.png) 

## Requirements

- Linux (not tested on other platforms)
- Python 3.6 or newer
- PyTorch >= 0.4.0
- CUDA >= 9.0
- cuDNN >= 7.1

## Getting started

Download or clone the full repository and run

```
python demo.py
```
